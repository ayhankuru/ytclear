## Install

```
npm install ytclear
```

## Usage


```js
var ytclear = require('ytclear');
```


```js

ytclear("Deadmau5 - Strobe (Lemony's Trap Remix) 2012");
// Deadmau5 - Strobe

```

## Or 

```sh
npm install -g ytclear

```
## Usage

```sh
echo "Deadmau5 - Strobe (Lemony's Trap Remix) 2012" | ytclear

//or 

ytclear "Deadmau5 - Strobe (Lemony's Trap Remix) 2012" 

```
