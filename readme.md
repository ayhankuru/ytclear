## Install

```
npm install @c0b41/ytclear
```

## Usage


```js
var ytclear = require('@c0b41/ytclear');
```


```js

ytclear("Deadmau5 - Strobe (Lemony's Trap Remix) 2012");
// Deadmau5 - Strobe

```

## Or 

```sh
npm install -g @c0b41/ytclear

```
## Usage

```sh
echo "Deadmau5 - Strobe (Lemony's Trap Remix) 2012" | ytclear

//or 

ytclear "Deadmau5 - Strobe (Lemony's Trap Remix) 2012" 

```
